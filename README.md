# Finite Difference Methods – Mathematical Foundations

Acest document prezintă fundamentele matematice complete ale metodei diferențelor finite aplicată pentru rezolvarea ecuațiilor diferențiale ordinare (1D) și cu derivate parțiale (2D).

## 📚 Conținut

Analizăm în detaliu algoritmii implementați în codurile Python, incluzând:

- Derivarea schemelor de diferențe finite  
- Teoria factorizării LU și metodele de substituție  
- Analiza convergenței și estimarea erorilor  
- Interpolarea Lagrange  
- Tratarea condițiilor de frontieră  

## 🧮 Domenii abordate

- Ecuații diferențiale ordinare (ODE – 1D)  
- Ecuații cu derivate parțiale (PDE – 2D)  

## ⚙️ Tehnologii utilizate

- Python  
- NumPy  
- Matplotlib (pentru vizualizări)

## 🎯 Obiectiv

Scopul acestui proiect este de a oferi o înțelegere riguroasă atât teoretică, cât și practică a metodelor numerice bazate pe diferențe finite.
